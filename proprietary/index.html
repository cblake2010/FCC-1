Raydar

Auto Production

Raydar Auto Production was created to track all mechanical testing data. It was expanded to track each quality inspection point and approvals. It also tracks all electrical tests are complete by tracking the test files created from our Wire Analyzer. The software continues to evolve today and servees as a cornerstone of our cable shop operation.

Backshell Toque

Backshell assemblies are torqued by technicians to settings gathered from MySQL backend. Current torque value are gathered through connected Torque Transducer connected via RS232.

Connection Settings

View where serial and database settings are modified and saved.

Employee Login

Each user has a login and unique pin verified through the database for login.

Employee Dashboard

Map to each area of the software, each use can choose any of these forms as a starting point as well, which is saved to the database and retrieved at login.

QA Admin

Determine and database each assessed quality check on every assembly.

Quality Assurance

Quality assuarance has a snap shot of all test data of an assembly, the needed quality checks, and saves checked off data back to the database
