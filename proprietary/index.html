SAIC

Project Development Database

The project development database was born out of a strong desire to eliminate redundant look ups. When initiated, each new project required much research into connectors, contacts, mating connectors, tooling, and many other items. This database utilized our researched data and continues to save countless dollars in saved labor. For this project I stood up an internal linux server and wrote the front end in HTML and JS. The backend consisted of PHP and a MySQL database.

Raydar

Auto Production

Raydar Auto Production was created to track all mechanical testing data. It was expanded to track each quality inspection point and approvals. It also tracks all electrical tests are complete by tracking the test files created from our Wire Analyzer. The software continues to evolve today and servees as a cornerstone of our cable shop operation.

The system was designed to run on a linux server and eventually Raspberry Pis at each technician bench as the system evolves so Linux felt to be a very good choice. It utilizes a MySQL database backend and the Gambas programming language, which is similar to Visual Basic.

Backshell Toque

Backshell assemblies are torqued by technicians to settings gathered from MySQL backend. Current torque value are gathered through a Torque Transducer connected via RS232.

Connection Settings

View and change serial and database connection settings. This makes it very easy to stand up multiple machines.

Employee Login

Each user has a login and unique pin verified through the database for login. Allowing us to track the technician responsible for mechanical measurements and even give us an idea of throughput.

Employee Dashboard

The system is comprised of several areas and the dashboard serves as the access point to each. Users can even choose their starting form if they find they consistently use the same area.

QA Admin

In the QA admin section we add all the key inspection points determined at project development. These will be required check boxes at the QA panel.

Quality Assurance

Quality assurance gets a snapshot of all gathered data for the assembly as well as the required inspection points. This gives the best overall view of an assembly and allows our QA to make informed decisions quickly and effectively.
